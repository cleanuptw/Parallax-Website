視差滾動效果

作者：Vineet Kumar

什麼是視差滾動？

視差滾動是一種網頁設計技巧，讓網站背景以比前景更慢的速度移動。隨著訪客向下滾動，會產生 3D 效果，增加景深並帶來更沉浸的瀏覽體驗。視差源自光學錯覺：人眼會覺得靠近的物體比遠處的物體大， 也會覺得遠處的物體移動得比較慢。這種錯覺早已被運用到各種媒介，創造更逼真的效果。最早可追溯 到迪士尼《白雪公主與七矮人》等傳統動畫，以及《超級瑪利歐》等電玩遊戲。隨著 CSS 和 HTML 的 進步，視差效果也發展成今天在網頁設計中常見的樣貌。

最早的瀏覽器視差滾動實作之一，是開發者 Glutnix 在 2007 年的部落格文章中分享的，包含以 JavaScript 和 CSS2 製作、支援 Internet Explorer 6 的範例程式碼與示範。但直到 2011 年 HTML5 和 CSS3 普及後，這項技巧才在網頁設計領域流行起來。支持者認為這是擁抱網路流動性的一個簡單方 式，也用視差背景吸引使用者並提升網站體驗。不過普渡大學在 2013 年發表的研究指出：「……雖然 視差滾動提升了使用者體驗的某些面向，但並不一定改善整體體驗」。


為什麼要使用視差圖形？

    設計師值得嘗試視差圖形的兩個主要原因：

  • 有助於說故事

    視差圖形很適合用來說故事。優秀的設計在於溝通——打造網站時，我們希望和訪客交流，向他們 傳達關於產品、服務或品牌的故事。視差滾動可以把訪客帶入獨特的線上體驗，甚至能在單頁 網站中呈現完整故事，讓使用者不用切換頁面就能了解你的品牌。

  • 能提升使用者參與度

    視差圖形能提升使用者參與度。降低網站跳出率是網頁設計的重要目標之一。跳出率衡量的是只 看一頁就按「返回」或關閉分頁的訪客比例。高跳出率會影響搜尋排名，因此任何能降低跳出率 的做法都值得嘗試。精心打造的視差頁面能帶來動態體驗，通常會讓使用者停留更久。


如果想在網站加入視差滾動效果，建議注意以下技巧：

  • 選擇性使用

    在特定元素使用視差滾動比在所有頁面全面套用效果更好。可以放在頁首、標題，或只放在首頁。 目標是讓視差滾動加強設計，而不是分散使用者注意力。

  • 圖片壓縮

    視差滾動會用到多種媒體檔和 CSS 形狀來營造景深。請避免使用檔案過大的圖片，以免影響載入 速度和使用者體驗。

  • 善用色彩

    視差滾動不只關乎前景與背景圖片，也和色彩有關。你可以在背景使用較低飽和度的色調，在前景 使用更鮮明的色彩，進一步加強景深效果。
